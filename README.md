# 🚗 EV Market Analysis Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing the Electric Vehicle (EV) market using Power BI. The dashboard provides insights into sales, pricing, performance, and battery efficiency.

## 🎯 Objective
To transform raw EV data into meaningful insights for better business decision-making.

## 📊 Dashboard Features
- Sales analysis by brand and segment
- Pricing trends across categories
- Performance analysis (horsepower, acceleration)
- Battery insights (range and efficiency)
- Interactive filters and slicers

## 💡 Key Insights
- Tesla leads in total EV sales
- Lucid has the highest average price and performance
- FWD dominates sales volume, AWD leads in performance
- Market shows clear segmentation between premium and budget

## 🛠️ Tools Used
- Power BI
- DAX
- Data Modeling
